# 🗓️ نظام إدارة الجداول الزمنية
### Shift Schedule Management System

**نظام متكامل لإدارة جداول الموظفين، الشيفتات، والطلبات**






</div>

***

## 📋 نبذة عن المشروع

نظام ويب متكامل لإدارة جداول عمل الموظفين مبني على ملف HTML واحد مع Firebase كقاعدة بيانات سحابية. يتيح للمديرين إدارة الجداول والشيفتات والأقسام، بينما يتيح للموظفين تقديم طلبات الإجازات وتبديل الشيفتات عبر بوابة خاصة.

***

## ✨ المميزات الرئيسية

### 👔 لوحة المدير
- **لوحة تحكم شاملة** — إحصائيات فورية عن الموظفين والتنبيهات والجداول
- **إدارة الجداول** — عرض وتعديل جداول الشيفتات شهرياً لكل قسم
- **إدارة الموظفين** — إضافة وتعديل وحذف بيانات الموظفين
- **إدارة الأقسام** — تحديد متطلبات التغطية الصباحية والمسائية
- **إدارة الشيفتات** — تخصيص أنواع الشيفتات وأوقاتها
- **الطلبات الواردة** — مراجعة وقبول أو رفض طلبات الموظفين
- **تنبيهات التغطية** — تحذيرات تلقائية عند نقص التغطية في الأقسام
- **تصدير واتساب** — إرسال الجدول الأسبوعي عبر واتساب أو PDF

### 👤 بوابة الموظف
- **جدول شخصي** — عرض جدول الشيفتات الشهري الخاص
- **طلب إجازة** — تقديم طلب إجازة بتاريخ محدد
- **تغيير شيفت** — طلب تغيير نوع الشيفت
- **تبديل مع زميل** — طلب تبديل الشيفت مع موظف آخر
- **تتبع الطلبات** — متابعة حالة الطلبات المقدمة
- **تغيير كلمة المرور** — تحديث كلمة المرور الشخصية

### 🎨 واجهة المستخدم
- دعم **الوضع الليلي والنهاري** تلقائياً
- واجهة **عربية RTL** بالكامل
- تصميم **متجاوب مع الجوال**
- دعم **PWA** — قابل للتثبيت كتطبيق على الجوال

***

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---|---|
| **HTML5 / CSS3 / JavaScript** | الواجهة الأمامية بالكامل |
| **Firebase Firestore** | قاعدة البيانات السحابية |
| **Firebase Authentication** | إدارة المستخدمين |
| **Lucide Icons** | أيقونات الواجهة |
| **Tajawal Font** | الخط العربي |
| **html2canvas + jsPDF** | تصدير PDF |

***
## ☁️ حدود Firebase المجانية

| الخدمة | الحد المجاني | يكفي لـ |
|---|---|---|
| Firestore قراءات | 50,000 / يوم | ~500 موظف |
| Firestore كتابات | 20,000 / يوم | ✅ أكثر من كافي |
| التخزين | 1 GB | سنوات من البيانات |
| Authentication | مجاني تماماً | ✅ |

## 📱 تثبيت كتطبيق (PWA)

**على الجوال (Android/iOS):**
```
افتح الموقع في المتصفح → القائمة → "إضافة إلى الشاشة الرئيسية"
```

**على الكمبيوتر (Chrome):**
```
افتح الموقع → شريط العنوان → أيقونة التثبيت ⊕
```
